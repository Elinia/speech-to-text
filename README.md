# speech-to-text

Minimal example of camera displaying and real-time speech to text in web app.

[SvelteKit ](https://kit.svelte.dev/)+ [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API)

## Installing

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```
